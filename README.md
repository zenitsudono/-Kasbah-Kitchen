# Kasbah Kitchen
![Page Screenshot](public/screenshot/Screenshot.png)

Kasbah Kitchen is a modern web application that allows users to search for recipes based on keywords, meal types, or ingredients. It utilizes the Spoonacular API to provide a list of recipes with details such as recipe name, preparation time, and cooking instructions. Users can also filter recipes by meal type with a beautiful, responsive interface.

## Technologies

Kasbah Kitchen is built with the following technologies:

- [ReactJS](https://reactjs.org/)
- [TailwindCSS](https://tailwindcss.com/)
- [React Query](https://react-query.tanstack.com/)
- [React Router](https://reactrouter.com/)

For detailed information on these technologies, please refer to their respective documentation.

## API

The recipe data is sourced from the [Spoonacular API](https://spoonacular.com/food-api). Please refer to their documentation for more information on how to use the API.

## Features

- **Modern UI Design**: Glass morphism effects, beautiful gradients, and smooth animations
- **Responsive Layout**: Optimized for all devices with mobile-first design
- **Recipe Search**: Search for recipes based on keywords, meal types, or ingredients
- **Category Filtering**: Filter food recipes by meal type with colorful category cards
- **Detailed Recipe Pages**: View complete recipe details including ingredients and step-by-step instructions
- **Modern Typography**: Beautiful fonts (Inter & Poppins) for enhanced readability
- **Interactive Elements**: Hover effects, micro-interactions, and smooth transitions

## Design Highlights

- **Color Palette**: Warm, food-inspired colors with primary orange, secondary green, and accent purple
- **Glass Morphism**: Modern glass effects throughout the interface
- **Custom Animations**: Smooth transitions and staggered animations for better UX
- **Enhanced Cards**: Modern recipe cards with hover effects and detailed information


## Installation

To use this application, you will need [Node.js](https://nodejs.org/) installed on your computer.

1. Clone the repository:
```bash
git clone https://github.com/zenitsudono/-Kasbah-Kitchen.git
cd -Kasbah-Kitchen
```

2. Install the dependencies:
```bash
npm install
```
3. Start the development server:
```bash
npm start
```
This will start the development server and open the application in your default browser
